## 故障排除
1. 如果程序无法启动，请检查Python版本是否>=3.8
2. 如果界面无法显示，请确认PyQt5安装正确
3. 如果浏览器功能异常，请运行: playwright install chromium
4. 数据库问题会在程序启动时自动修复
