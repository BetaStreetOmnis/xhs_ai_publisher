# 小红书从 0 到 1 教程（选题 → 文案 → 封面 → 发布 → 运营）

这份文档面向想在小红书稳定输出笔记的新手/进阶创作者，覆盖「选题、写法、封面、发布、运营复盘」的核心流程，并补充如何搭配本项目 `xhs_ai_publisher` 提效。

> 提醒：请遵守小红书平台规则与法律法规；内容发布前务必人工审核，避免敏感/虚假/侵权信息。本文只提供通用创作方法与工具使用建议，不鼓励滥用自动化。

---

## 0. 小白快速上手（用 `xhs_ai_publisher` 发第一篇）

如果你只想最快发出第一篇笔记，先按这 4 步走；后面的「选题/标题/封面/运营」再慢慢学也来得及。

### 0.1 下载安装到电脑（不用懂技术）

1. 打开项目页面（GitHub 或 Gitee）。
2. 点击下载：`Code` → `Download ZIP`（或「下载 ZIP」）。
3. 解压后，进入文件夹 `xhs_ai_publisher`（你会看到 `install.bat` / `install.sh` / `启动程序.bat` / `启动程序.sh`）。

### 0.2 一键安装（Windows）

1. 双击 `install.bat`（第一次会下载依赖，耐心等它跑完）。
2. 看到提示 `🎉 Done.` 后，双击 `启动程序.bat`。
3. 如果安装失败：优先安装 **Python 3.11/3.12（64 位）**，然后重新双击 `install.bat`。

### 0.3 一键安装（macOS）

1. 先安装 **Python 3.11/3.12**。
2. 打开「终端 Terminal」→ 输入 `cd `（cd 后面有个空格）→ 把 `xhs_ai_publisher` 文件夹直接拖进终端窗口 → 回车。
3. 执行（复制粘贴即可）：
   - `chmod +x install.sh 启动程序.sh`
   - `./install.sh`
4. 安装完成后，执行：`./启动程序.sh`

### 0.4 软件内发第一篇（照着点就行）

1. **登录**：输入手机号 → 获取验证码 → 登录成功后会自动保存状态。
2. **写主题**：首页输入主题（例如：`打工人一周减脂晚餐` / `油痘肌护肤流程` / `新手穿搭公式`）。
3. **生成文案**：选择一个「文案模板」（不懂就先用默认）→ 点「生成内容」→ 把生成结果改成你的口吻（加上你的真实经验/参数/价格/踩坑）。
4. **做封面/多图**：侧边栏点「🖼️ 封面中心」→ 选模板 → 「✅ 应用到首页」→ 生成后在首页点「📥 下载图片」。
5. **预览发布**：点「预览发布」确认无误 → 点「发布」；需要无人值守就用「⏰ 定时发布」（程序必须一直开着，电脑不要睡眠）。

## 1. 账号准备（定位先于内容）

- **垂直领域（垂类）**：选择你能长期输出的方向（护肤/穿搭/健身/家居/学习/职场/本地生活/母婴等），避免“什么都发”。
- **人设与受众**：一句话说明你是谁、能帮谁解决什么问题（例如「油痘肌成分党」「预算穿搭」「健身新手减脂」）。
- **主页三件套**：头像清晰、昵称好记、简介包含关键词（便于搜索）。
- **内容目标**：种草转化 / 引流咨询 / 品牌曝光 / 建立专业度，不同目标决定写法与发布节奏。

## 2. 选题方法（从“我想写”变成“大家想看”）

### 2.1 三个最稳的选题来源

1. **平台搜索**：在小红书搜索框输入核心词，记录联想词/热门笔记标题结构。
2. **同赛道对标**：找 10 个同领域账号，拆解他们“爆款笔记”的共同点（标题、封面、结构、话题、评论区问题）。
3. **热点借势**：结合当日热榜/节日节点/季节变化，把热点落到你的垂类里（不要硬蹭）。

### 2.2 搭配 `xhs_ai_publisher` 的选题入口

- 侧边栏「📊 数据中心」查看微博/百度/头条/B站等热榜 → 选中热点后点「✍️ 用作首页主题」一键带回生成。

## 3. 标题与开头（决定点击率）

### 3.1 标题常用公式（可直接套）

- **痛点 + 结果**：`油痘肌这样做，3 天闭口少一半（亲测）`
- **人群 + 方案**：`打工人晚餐怎么吃？这份一周减脂模板抄作业`
- **对比 + 结论**：`XX 和 XX 到底差在哪？买前一定要看这一条`
- **数字清单**：`我总结了 7 个新手最容易踩的坑（第 3 个最多）`
- **场景化**：`第一次去重庆怎么安排？按这个路线不走冤枉路`

建议：
- **标题尽量“具体”**（人群/场景/结果/时间/预算/对比维度）。
- 避免夸张与虚假承诺；不要堆砌过多符号。

### 3.2 开头 3 行（先抓住注意力）

三选一即可：
- **直接给结果**：先给结论/对比表，再解释原因。
- **抛出痛点**：你是不是也遇到…？（让读者对号入座）
- **设置悬念**：我试了 3 种方法，只有 1 种真的有效…

## 4. 正文结构（让读者愿意看完并收藏）

### 4.1 万能结构

- **背景/问题**：为什么要写这篇？适用谁？
- **步骤/要点**：按 3–7 条分点写清楚（每点尽量短句）。
- **避坑/注意事项**：把“别人会问的问题”提前回答。
- **清单/总结**：结尾做一个可收藏的 checklist。
- **互动引导**：评论区提问（例如“你更关心 A 还是 B？”）。

### 4.2 排版小技巧

- 句子短一点；多用列表/小标题；少用大段长文。
- 关键词可用「加粗」或「分隔线」突出（不要过度花哨）。
- 图片与文字对齐：图讲感受/对比，字讲步骤/理由/参数。

### 4.3 标签/话题怎么选

- 以“**精准相关**”为第一原则，避免无关标签堆砌。
- 组合方式：`1-2 个大词 + 3-6 个细分词 + 1-2 个场景词`。
- 长期做同一垂类，标签也要相对稳定（利于系统识别你的领域）。

## 5. 封面与配图（决定停留与转发）

### 5.1 封面原则

- **一眼能读懂**：封面字尽量少（8–14 个字更稳），对比强烈。
- **一句话讲清价值**：结果/清单/对比/路线/配方，选一种即可。
- **风格统一**：长期用相似版式与色系，建立“品牌感”。

### 5.2 多图内容图

- 第一张做“结论/亮点”，后面按步骤展开。
- 用「编号/流程」串起来，读者更愿意滑完。

### 5.3 搭配 `xhs_ai_publisher` 的封面能力

- 侧边栏「🖼️ 封面中心」选择模板 → 「✅ 应用到首页」。
- 生成内容时可自动输出 **封面 + 多页内容图**，并在首页「📥 下载图片」导出。

## 6. 发布策略（节奏比“勤奋”更重要）

- **频率**：新号建议先稳定在 `每周 3–5 篇`，比“连发”更安全也更容易复盘。
- **时间**：根据你赛道的目标人群作息测试（午休/晚间通常更集中），关键是固定并观察数据变化。
- **内容比例**：`干货/经验` 与 `种草/推荐` 组合输出，避免只发广告。
- **定时发布**：需要无人值守时再用；优先保证内容质量与合规。

## 7. 运营与复盘（用数据迭代，而不是凭感觉）

### 7.1 关注哪些指标

- 曝光 → 点击 → 互动（赞/藏/评）→ 关注
- 收藏与评论往往比点赞更能反映“内容价值”

### 7.2 一页复盘模板（每篇都能用）

- 这篇的**选题**来自哪里？（搜索/对标/热点/用户提问）
- **标题**用了什么公式？点击是否达标？
- **封面**是否清晰表达价值？是否需要优化关键词？
- 评论区出现了哪些高频问题？下一篇可以回答什么？

## 8. `xhs_ai_publisher` 更详细的操作说明（小白版）

### 8.1 先把「账号」和「环境」搞清楚

- 侧边栏「👥 用户」：可以新增/切换/删除用户（不同用户的数据会隔离保存）。
- 侧边栏「🌐 浏览器环境（可选）」：如果你需要代理/指纹，先创建环境并设为「⭐ 默认环境」，发布时会自动应用。

### 8.2 选题（不知道写啥就用这两个入口）

- **直接写**：首页主题输入框里写一句话即可（越具体越好：人群/场景/预算/目标）。
- **热点带回**：侧边栏「📊 数据中心」→ 选中热点 → 点击「✍️ 用作首页主题」。

### 8.3 生成文案（生成后一定要“二改”）

1. 在首页选择「文案模板」（不懂就先用默认）。
2. 点击生成标题/内容。
3. **人工二改**（强烈建议）：
   - 把“你真的做过的细节”补进去（步骤/价格/时间/效果/对比）。
   - 删除夸张承诺与敏感表述。
   - 用你自己的口头禅/语气，让它像“真人笔记”。

### 8.4 封面/多图（让读者一眼看懂）

1. 侧边栏「🖼️ 封面中心」→ 选择模板 → 点击「✅ 应用到首页」。
2. 生成后在首页右侧预览区确认清晰度。
3. 点击「📥 下载图片」把封面 + 多页内容图导出。

### 8.5 发布与定时发布（最容易踩坑的地方）

- **发布前**：先点「预览发布」检查标题/话题/图片顺序/错别字。
- **定时发布**：必须保持软件一直运行、电脑不休眠、网络稳定；否则到点可能发不出去。
- **触发验证/风控**：降低发布频率、提高内容差异化、不要重复搬运；必要时改为手动发布。

## 9. 常见问题（写给第一次发的人）

- **登录/发布时出现验证**：这是平台的正常风控策略；建议降低发布频率、保证内容差异化、稳定网络环境，必要时改为人工发布与核验。
- **图片上传失败**：检查图片格式（PNG/JPG）、尺寸与体积；封面字不要太小。
- **内容被提示违规/限流**：先排查敏感词、夸大承诺、搬运/侵权；避免同质化重复发布。
- **模型配置不生效**：检查「⚙️ 后台配置」里的模型与 Key 是否正确；Key 会加密保存到 `~/.xhs_system/keys.enc`。

## 10. 可复制的笔记模板（直接粘贴改）

### 10.1 清单型（收藏友好）

- 标题：`我整理了 {N} 个 {主题} 清单（新手直接抄）`
- 正文：
  1) 适用人群：…  
  2) 结论先给：…  
  3) 清单 1–N：…  
  4) 避坑：…  
  5) 结尾提问：你还想要哪个版本？

### 10.2 对比型（转化友好）

- 标题：`{A} vs {B}：差别在这 {N} 点（选错很亏）`
- 正文：对比维度（价格/效果/成分/适用场景/优缺点）→ 最后给推荐人群结论

### 10.3 路线/步骤型（实用友好）

- 标题：`{地点/目标} 一次搞定：按这个 {N} 步走就行`
- 正文：步骤 1–N（每步：时间/预算/要点/注意事项）→ 结尾附「一图总结」
